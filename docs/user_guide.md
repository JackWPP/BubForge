# 用户手册

## 1. 创建/选择项目
启动后点击“选择项目”，指定一个目录作为项目根目录。系统会自动创建标准目录结构。

## 2. 打开视频
点击“打开视频”选择本地视频文件。视频会被登记到 `sources.csv`。

## 2.1 布局与自适应
- 采用可拖拽工作区（Dock）：
  - 时间线在底部
  - 素材选择在左侧
  - 导出面板在右侧
- 你可以直接用鼠标拖动每个面板标题栏，把面板停靠到任意边或浮动成独立窗口
- 菜单“视图”可单独显示/隐藏这些面板
- “重置工作区”可恢复默认布局
- 工作区会自动保存，下次打开保持上一次布局
- “比例”可切换预览图像显示方式：比例锁定/自由拉伸
- 顶部“显示”可切换视频缩放（适配/100%/150%/200%）

## 3. 预览与标记
- Space：播放/暂停
- A/D：逐帧后退/前进
- J/L：快退/快进（约 1 秒）
- 左/右方向键：1 帧步进
- Shift + 左/右方向键：10 帧步进
- Ctrl + 左/右方向键：约 1 秒步进
- Ctrl + Shift + 左/右方向键：约 5 秒步进
- I/O：标记 In/Out
- Enter：添加区间

## 4. 保存关键帧
按 S 或点击“保存关键帧”，当前帧将保存到 `frames/<video_folder>/keyframes/`，并写入 `metadata/frames.csv`。
保存后会自动跳到下一帧，便于快速连续挑选关键帧。

## 5. 区间抽帧
按 E 开始区间抽帧，默认 5fps。帧保存到 `frames/<video_folder>/ranges/`。
当 In/Out 同时存在时，区间会自动加入“区间”列表并可在导出时使用。

## 5.1 时间线标记
- 已设置的 In/Out 会显示在时间线上
- 已保存关键帧会显示为时间线刻度标记

## 6. 导出数据集骨架
选择导出格式并点击“开始导出”，生成 Raw/Ultralytics/COCO 骨架。
